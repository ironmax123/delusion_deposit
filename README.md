# 妄想貯金
## 仕様
    1. 基本の外食の食費を設定してもらう
    2. 外食したときだけボタンを押す 
    3. 一週間ずつ貯金されていく
    4. 外食したときに入力
## 画面構成
1. ホーム 
    - 現在の貯金額
    - 期限
    - 目標金額
    - 目標までの期限
    - 達成おめでとう
    - 使用金額入力ボタン => Bottom_Sheet

2. 目標入力画面
    - 達成目標の入力
    - 目標金額の入力
    - 期限の入力（yy/mm/dd）
    ### 要件
    - 目標がない場合、目標設定画面へ遷移
    - 目標は最初に決めたら変えれない
    - 達成したら、目標及び履歴を消す

3. 基準入力画面
    - 基準の設定（週いくら外食で使うか）

4. 外食の履歴画面
    - リストで日付と金額が表示される
